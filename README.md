Resumen del Proyecto "Mente Clara"


¿De qué va la web?


Es un sitio web muy sencillo, hecho solo con HTML5 y CSS3 .

Quiero concienciar a la gente y darles información real sobre enfermedades mentales, para que se entiendan mejor.

También hay una sección con enlaces y recursos útiles por si alguien necesita ayuda.

Hice un esfuerzo para que se vea bien en el móvil y para que fuera lo más fácil de usar posible (lo de "diseño accesible" y "semántica HTML5").


¿Para quién es?


La idea es que le sirva a:

Cualquier persona (adolescentes y adultos).

Familiares o amigos que tengan a alguien con problemas mentales.

Estudiantes o gente que le interese mucho el tema.


¿Qué usé para hacerlo?


HTML5: Para poner el contenido y la estructura.

CSS3: Para que se vea bonito y se adapte a diferentes tamaños de pantalla.

Git y GitHub: Para ir guardando los cambios y tener el proyecto online.

Lo publiqué en GitHub Pages para que cualquiera pueda verlo en un enlace.


¿Cómo está organizado?


La web tiene las siguientes páginas:

index.html: La página de inicio.

trastornos.html: Una página sobre algunos trastornos comunes.

recursos.html: Donde están los enlaces de ayuda.

Todos los estilos están en la carpeta css/styles.css y las imágenes en la carpeta images/.


Instalación (¡Para verlo en tu PC!)


Si quieres ver el código en tu ordenador, es fácil:

Copia el repositorio con git clone ....

Entra en la carpeta del proyecto con cd ....

Abre el archivo index.html en tu navegador y ¡listo!.
